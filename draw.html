Winner Selection

This prize draw is conducted in a fair, transparent, and random manner.

Draw Method: Random selection using random.org

How the Winner Is Chosen

  • All valid paid entries are collected in a participant list
  • The draw is conducted live on social media
  • The participant list is randomized using random.org
  • The first entry selected is declared the winner

Live Draw

The draw will take place live on our official social media channels. The date and time of the draw will be announced in advance.

Winner Contact

  • The winner will be contacted using the phone number provided
  • The winner must respond within 48 hours
  • If no response is received, a new draw will take place

All entries have an equal chance of winning.
